# 04_Single_Neuron_Network_with_and_without_PCA_Equipo_3.ipynb

Este repositorio contiene el notebook **`04_Single_Neuron_Network_with_and_without_PCA_Equipo_3.ipynb`**, el cual implementa y compara un **modelo de red neuronal de una sola neurona** con y sin **PCA (Principal Component Analysis)**.

## Requisitos

Antes de ejecutar el notebook, asegúrate de tener:

- **Python** instalado (se recomienda Python 3.8 o superior)
- **Jupyter Notebook** o **JupyterLab**

También se recomienda tener instaladas las siguientes librerías comunes de ciencia de datos:

```bash
pip install numpy pandas matplotlib scikit-learn jupyter
